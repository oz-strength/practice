웹 개발

웹 개발은 웹사이트나 웹 애플리케이션을 만드는 과정입니다. HTML, CSS, JavaScript와 같은 기술을 사용하여 사용자 인터페이스를 구축하고, 서버 측 언어와 데이터베이스를 통해 동적인 기능을 구현합니다.

프론트 엔드

프론트 엔드는 사용자가 직접 상호작용하는 웹사이트의 부분을 의미합니다. HTML, CSS, JavaScript를 사용하여 웹페이지의 구조, 스타일, 동작을 정의합니다.

백엔드

백엔드는 서버, 데이터베이스, 애플리케이션 로직을 포함한 웹사이트의 보이지 않는 부분을 의미합니다. 서버 측 언어 (예: Python, Ruby, PHP)를 사용하여 데이터 처리와 비즈니스 로직을 구현합니다.
